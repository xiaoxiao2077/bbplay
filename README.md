# BBPlay Flutter

BBPlay 是一个第三方的 Bilibili 播放器应用，专为儿童设计，提供安全、简洁的视频观看体验。该应用支持移动设备和桌面设备，具有完整的视频播放、历史记录、收藏等功能。

## 功能特性

- **视频播放**：支持 Bilibili 视频播放，包括横屏和竖屏模式
- **历史记录**：自动记录观看历史
- **收藏功能**：收藏喜欢的视频
- **时间分段**：按时间段查看视频内容
- **搜索功能**：支持视频搜索
- **登录功能**：支持通过二维码登录 Bilibili 账号
- **多平台支持**：同时支持移动设备（Android/iOS）和桌面设备（Windows/macOS）
- **AI 总结**：提供视频 AI 智能总结功能
- **字幕支持**：支持视频字幕显示
- **播放控制**：播放速度调节、进度控制等功能

## 技术栈

- Flutter 3.x (SDK >=3.0.0 <4.0.0)
- Dart
- GoRouter (页面路由)
- MediaKit (视频播放器)
- Provider (状态管理)
- Dio (HTTP 客户端)
- SQLite (本地数据库)

## 环境要求

- Flutter SDK: >=3.0.0 <4.0.0
- Dart SDK: >=3.0.0 <4.0.0
- 支持平台：Android, iOS, Windows, macOS
